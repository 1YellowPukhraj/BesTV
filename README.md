# BesTV
Android TV App powered by TMDb. 
It is a easy way to find the best TV content, the top movies, series... all of that in your TV.

# Implementations
<ul>
<li>The Movie Database (TMDb)</li>
<li>Dagger 2
<li>Retrofit 2
<li>RxJava 2  
<li>Glide
</ul>
